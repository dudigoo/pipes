🎯 מבוא

מערכת Pipeline Inspector היא תוכנה מקצועית לניתוח וניהול בדיקות צנרת. המערכת מיועדת למהנדסים המבצעים בדיקות רובוט בצנרת ביוב ומים.

יכולות עיקריות:

  • ניהול פרויקטים - יצירה ועריכה של פרויקטי בדיקה
  • תיעוד משתתפים - ניהול לקוחות, מנהלים ומפעילים
  • ניתוח קטעי צנרת - תיעוד מפורט של כל קטע
  • רישום אירועים - תיעוד ליקויים ובעיות שנמצאו
  • ניתוח וידאו - צפייה ובדיקה של וידאו מהרובוט
  • יצירת דוחות - הפקת דוחות PDF מקצועיים

🚀 התחלת עבודה

רישוי המערכת

חשוב: המערכת דורשת רישיון תקף לפני השימוש!
  1. הרץ את התוכנה לראשונה
  2. העתק את מזהה החומרה (Hardware ID) שמוצג
  3. שלח למחלקת התמיכה לקבלת רישיון
  4. הכנס את קוד הרישיון שהתקבל

הפעלת המערכת

  1. פתח את קובץ pipe_inspect.exe
  2. המתן להטענת המערכת
  3. יוצג המסך הראשי עם רשימת הפרויקטים

📋 ניהול פרויקטים

יצירת פרויקט חדש

  1. לחץ על כפתור פרויקט חדש במסך הראשי
  2. מלא את הפרטים הנדרשים:
שדה תיאור חובה
שם הפרויקט שם מזהה לפרויקט ✅ כן
מספר פרויקט מספר פנימי לזיהוי ❌ לא
תיאור הפרויקט תיאור כללי ❌ לא
תאריך התחלה פורמט: יום/חודש/שנה ❌ לא
תאריך סיום פורמט: יום/חודש/שנה ❌ לא
עיר ואתר מיקום הפרויקט ❌ לא

הוספת תמונה לפרויקט

  1. במסך עריכת הפרויקט, לחץ על סימן ה+
  2. בחר תמונה מהמחשב
  3. התמונה תוצג כתמונה קטנה
  4. ניתן לסובב או למחוק התמונה

פעולות זמינות בפרויקט:

  • משתתפים - ניהול צוות הפרויקט
  • קטעים - ניהול קטעי הצנרת
  • הדפסה - יצירת דוח מקצועי

👥 ניהול משתתפים

סוגי משתתפים

המערכת מחלקת משתתפים לשלושה סוגים:
  • לקוחות - בעלי הפרויקט
  • מנהלים - מנהלי הפרויקט
  • מפעילים - מבצעי הבדיקה

הוספת משתתף חדש

  1. במסך הפרויקט, לחץ משתתפים
  2. בחר את הסוג המתאים (לקוח/מנהל/מפעיל)
  3. לחץ הוסף
  4. מלא את הפרטים הנדרשים

פרטי משתתף:

  • שם (חובה)
  • חברה
  • טלפון ראשי ומשני
  • כתובת אימייל
  • כתובת: רחוב ועיר

הוספת לוגו לחברה

  1. במסך עריכת משתתף, לחץ העלה לוגו
  2. בחר קובץ תמונה
  3. הלוגו יוצג ויישמר עם פרטי החברה

שיוך משתתפים לפרויקט

  1. במסך המשתתפים, סמן ✓ ליד המשתתפים הרלוונטיים
  2. לחץ שמור כדי לשייך למפרויקט

🔧 ניהול קטעים

המסך מחולק לארבעה חלקים:
  1. רשימת קטעים (שמאל עליון)
  2. מעקב אירועים (שמאל תחתון)
  3. גרף הצנרת (ימין עליון)
  4. נגן וידאו (ימין תחתון)

יצירת קטע חדש

  1. במסך הקטעים, לחץ הוסף קטע
  2. מספר הקטע יוקצה אוטומטית
  3. מלא את הפרטים הנדרשים

פרטי קטע בסיסיים:

  • מספר צנרת (ברירת מחדל: תאריך ושעה נוכחיים)
  • תאריך ושעת תחילה
  • אורך משוער (במטרים)
  • בור עליון ותחתון

מאפייני צנרת:

מאפיין אפשרויות
צורת צינור עגול, ביצה, מרובע
קוטר במילימטרים
חומר צינור בטון, פלסטיק, מתכת וכו'
שימוש מים (כחול), ביוב (אדום)

פרטי מיקום:

  • רחוב, עיר, אתר
  • עומק התחלה וסיום
  • כיוון זרימה

הוספת וידאו לקטע

  1. במסך עריכת הקטע, לחץ בחר וידאו
  2. בחר את קובץ הוידאו מהמחשב
  3. הוידאו יוצג בנגן בחלק התחתון ימין
  4. נתיב הקובץ נשמר במסד הנתונים

📝 רישום אירועים

הוספת אירוע חדש

  1. בחר קטע ברשימת הקטעים
  2. לחץ הוסף אירוע מתחת לטבלת המעקב
  3. ייפתח חלון רישום אירוע עם שלושה חלקים

חלק א': בחירת קוד אירוע

  • רשימה ממוינת של כל קודי האירועים
  • שדה חיפוש למציאת קוד מהירה
  • תצוגה של: מספר קוד + סוג + תיאור

חלק ב': פרטי האירוע

שדות חובה לכל אירוע:
  • מרחק בוידאו (במטרים מתחילת הקטע)
  • זמן בוידאו (יתמלא אוטומטית מהנגן)
  • חומרות (0-5)
  • הערות

שדות נוספים לפי סוג האירוע:

סוג אירוע שדות נוספים
אירועי מבנה
(סדקים, שברים)		 • שעון התחלה וסיום
• אורך ורוחב הליקוי
אירועי תפעול ותחזוקה • אורך הליקוי
• אחוזי חסימה
• מקור הבעיה
אירועי צומת • שעון הצומת
• רוחב הצומת
• פעיל/לא פעיל

חלק ג': נגן וידאו

  • נגן משולב לצפייה בזמן רישום האירוע
  • כפתור צילום מסך לשמירת תמונה
  • התמונה שמורה במסד הנתונים

עריכת אירועים קיימים

  1. לחץ פעמיים על אירוע ברשימת המעקב
  2. ערוך את הפרטים הנדרשים
  3. לחץ שמור

🎥 צפייה בווידאו

נגן הוידאו המשולב כולל:

  • הפעלה/השהיה
  • חזרה 2 שניות אחורה
  • סרגל ניווט לדילוג לזמן מסוים
  • תצוגת זמן נוכחי וכולל
  • מרחק מתצוגת הוידאו (פינה ימנית עליונה)

שימוש בנגן:

  1. בחר קטע עם וידאו מרובץ
  2. הוידאו יוטען אוטומטית במצב השהיה
  3. השתמש בכפתורי הבקרה לניווט
צפייה במסך מלא:
  1. לחץ על כפתור מסך מלא בנגן
  2. הוידאו ייפתח בחלון נפרד
  3. לחץ X לחזרה למסך הראשי

ניווט לאירועים:

  • בחירת אירוע מהרשימה תזיז את הוידאו לזמן האירוע
  • קו ירוק יופיע בגרף הצנרת במיקום האירוע

📄 יצירת דוחות

הפקת דוח PDF

  1. במסך הפרויקט, לחץ הדפסה
  2. המערכת תיצור דוח מקצועי

תוכן הדוח:

סוג עמוד תוכן
עמוד פתיחה פרטי הפרויקט הכלליים
פרופיל קווי סיכום כל הקטעים לפי קוטר
עמודי קטעים עמוד נפרד לכל קטע עם:
• פרטי הקטע המלאים
• גרף הצנרת עם סימון אירועים
• טבלת אירועים מפורטת
טבלת אירועים מלאה רשימה מפורטת של כל האירועים
צילומי מסך כל התמונות שצולמו במהלך הבדיקה
הדוח כולל:
  • תצוגה בעברית (מימין לשמאל)
  • לוגו החברה המבצעת
  • פרטי איש הקשר
  • דירוגי חומרה לכל אירוע
  • מידע טכני מפורט

⚙️ הגדרות מערכת

קבצי תצורה

המערכת משתמשת בקבצי JSON לתצורה

config/config.json - הגדרות כלליות:

{
  "app_language": "he",
  "default_date_format": "dd/mm/yyyy",
  "video_extensions": [".mp4", ".avi", ".mov"],
  "image_extensions": [".jpg", ".jpeg", ".png", ".bmp"]
}

config/selection.json - רשימות בחירה:

  • קודי אירועים ותיאוריהם
  • סוגי צינורות וחומרים
  • רמות חומרה

שינוי שפה

  1. ערוך את config/config.json
  2. שנה את app_language ל-"en" (אנגלית) או "he" (עברית)
  3. הפעל מחדש את התוכנה

🛠️ פתרון בעיות

בעיות נפוצות ופתרונות:

🎥 הוידאו לא מתנגן:

  • ודא שקובץ הוידאו קיים במיקום המקורי
  • בדוק שפורמט הוידאו נתמך (.mp4, .avi, .mov)
  • נסה להזיז את הקובץ למיקום אחר ולעדכן בהגדרות הקטע

💾 שגיאות במסד הנתונים:

  • ודא שיש הרשאות כתיבה לתיקיית התוכנה
  • בדוק שקובץ pipes.db לא פתוח בתוכנה אחרת
  • במקרה של קורפציה, השתמש בקובץ הגיבוי pipes.db.backup_*

🔐 בעיות ברישיון:

  • ודא שהרישיון תקף ולא פג
  • בדוק חיבור לאינטרנט לוודא רישיון שנתי
  • צור קשר עם התמיכה לחידוש רישיון

🖥️ בעיות תצוגה:

  • אם הטקסט מוצג הפוך, ודא שהשפה מוגדרת כ-"he"
  • במקרה של גופנים לא ברורים, ודא שקובץ DejaVuSans.ttf קיים

🖨️ בעיות הדפסה:

  • ודא שכל הנתונים מלאים לפני יצירת הדוח
  • בדוק שיש מקום פנוי בדיסק לקובץ ה-PDF
  • נסה לסגור תוכנות אחרות שעשויות לחסום יצירת קבצים

קבלת תמיכה

למקרה של בעיות שלא נפתרו:
  1. צלם מסך של השגיאה
  2. שמור את קובץ app.log (אם קיים)
  3. צור קשר עם צוות התמיכה עם הפרטים הבאים:
    • תיאור הבעיה
    • שלבים לשחזור הבעיה
    • צילום מסך
    • מזהה החומרה (Hardware ID)
